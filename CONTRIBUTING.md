# Contributing to Amethyst Launcher
## Welcome!
Thanks for your interest in contributing! Read the following guidelines before getting started.

## Reporting Bugs
Report a Github Issue with the following contents:
+ Operating System and Version
+ Geometry Dash version
+ Steps to reproduce
+ Logs or error messages

## PRs
+ Keep it focused on one thing at a time
+ Make sure it builds and runs
+ Describe what your PR does and why
+ If it breaks functionality, explain why

### For bigger requests
+ Make sure it's compatible with existing instances
+ If not, make some kind of way to convert it
+ Keep it clean, we won't accept one big file

## What we are looking for
+ Bug Fixes
+ Platform Improvements
+ General Improvements
+ Adding features from the [ROADMAP.md](ROADMAP.mds)

## Code Style
+ Keep it readable and consistent with surrounding code

## Don't do this!
+ Adding code that bypasses Steam, or removing code that requires it
+ Anything piracy related
+ Anything that breaks existing functionality for no reason

## Building
+ Read the building section in [README.md](README.md) for steps to build the app
